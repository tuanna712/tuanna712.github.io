E&P knowledge sharing with RAG chatbot and knowledge graph integration (2023)

A Retrieval Augmented Generation (RAG) chatbot system for knowledge sharing in the petroleum exploration and production domain, powered by over 2,000 documents, including books and research reports.

Abstract

Over 40 years of research and development, the Vietnam Petroleum Institute has accumulated vast petroleum knowledge. To meet the growing demand for knowledge sharing, a chatbot system has been developed to optimize and access this collected data in Vietnamese, English, and Russian.

The solution utilizes a RAG chatbot system, which requires processing and organizing domain-specific knowledge. The system is integrated with a web-based interface, allowing the RAG chatbot to answer user questions while providing references from raw sources, including both text and image references.

This solution ensures flexibility in chatbot responses while maintaining authentication and accuracy of the source references.

Key Results

  • Unstructured data processing (pdf, doc, txt, images,...) to generate a uniformed database.
  • RAG chatbot system to interact with domain knowledge
  • Web-app interface for chatbot and references.
  • Web-app interface for documents preview and sharing.

Platforms/ Frameworks

  • Data storage: MS Sharepoint
  • SQL database: MySQL
  • Vector database: Weaviate
  • Web front-end: HTML, CSS, JS
  • Web back-end: Django
  • Chatbot: OpenAI, Semantic Kernel
