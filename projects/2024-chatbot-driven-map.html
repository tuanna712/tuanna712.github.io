Chatbot Driven Map navigation with integrated graph database (2024)

An automatically updated map-based website powered by a graph database and chatbot interactions, dynamically navigating to provide quick and accurate responses with relevant object displays.

Abstract

This project develops an innovative map-based website powered by a graph database and integrated chatbot interactions, offering users an intuitive navigation experience.

The core of the project is a graph database and a reasoning system designed for LLMs, leveraging graph connections to provide efficient interdisciplinary knowledge queries.

In line with current trends, a RAG-based chatbot is developed for domain-specific question answering, navigating the map to deliver accurate location and object information.

Overall, the project enhances the user experience by facilitating smooth knowledge exploration and sharing through dynamic, context-aware interactions.

Objectives

  • Develop a knowledge graph approach for data structure
  • Implement an LLM chatbot for domain-specific question answering, interacting with private data
  • Create an automatically and dynamically updating web map that navigates based on chatbot topics and displays additional information for map objects

Key Results

  • An interdisciplinary domain graph database integrated with spatial objects
  • A map-based website that navigates based on user conversation and related objects
  • A chatbot system that reasons through graph connections

Main Tasks (Personal)

  • Design graph database schema in Neo4j
  • Develop tools for constructing the graph database
  • Organize map objects and provide spatial information for nodes in the graph database
  • Structure domain knowledge and link it to the graph for an efficient RAG chatbot system
  • Develop a RAG chatbot system using the graph database
  • Design APIs for map navigation during chatbot interactions

Platforms/ Frameworks

  • Graph database: Neo4j
  • Chatbot: OpenAI, Semantic Kernel
  • REST API
