Resume Analysis (2024)

An analysis of over 12,000 resumes and 2,000 job descriptions based on real-world data, focusing on resume classification and resume-job matching. This project is part of the Data Mining (CSCI 5502) class at CU Boulder, Fall 2024.

Abstract

The Resume Analysis project aims to develop a universal resume parser that extracts key information from resumes, addressing challenges posed by diverse formats. By aligning candidates with suitable job roles based on job descriptions and industry standards, the system enhances recruitment efficiency and reduces bias.

The project focuses on two main tasks: Resume Classification and Resume-Job Matching, utilizing both machine learning (ML) and deep learning (DL).

The best model achieved 82% performance, and the final findings offer valuable insights into resume analysis.

Objectives

  • Handle data scraping skills from real-world resources
  • Capture data nature and importance for preparation
  • Undergo the data science life cycle to solve real-world problems
  • Develop and evaluate multiple ML and DL models for prediction tasks
  • Develop a website to report project steps
  • Write an ACM-formatted report to capture insights

Key Results

  • Successfully scraped over 12,000 resumes and 2,000 job descriptions from PostJobFree.com
  • Achieved 82% accuracy in resume classification
  • Answered 10 priori questions using the data proofs
  • Developed a website and report for the final presentation

Main Tasks

This is a team project (3 members, equally contributed). The main tasks I handled include:

  • Web scraping
  • Developing and evaluating ML models for resume classification
  • Resume and job matching using the TF-IDF algorithm
  • Reporting in ACM format

Platforms/ Frameworks

  • Web scraping: BeautifulSoup
  • Text processing: NLTK, Spacy, Re
  • Machine learning: Sci-kit learn
  • Deep learning: Keras
