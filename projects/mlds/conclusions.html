Conclusions

Started with the machine learning approaches for Lithology Classification using borehole data, the project has collected and prepared a dataset from two different sources: USGS API and FORCE ML 2020.

Results
Using the Unsupervised Learning approach, the project has clearly demonstrated the potential of clustering techniques in identifying distinct lithological groups based on well log data.

The results indicate that the clustering algorithms effectively captured the underlying patterns in the data, which are most likely related to the major classes of lithology.

The Supervised learning approaches showed promising results, with the Random Forest model achieving the highest accuracy of 0.95. It indicates that the model can effectively classify lithological types based on well log data.

Inferences
The project has successfully demonstrated the feasibility of using machine learning techniques for lithology classification, providing a valuable tool for geoscientists and engineers in the field of geology and petroleum engineering. The results can be used to improve the understanding of subsurface geology, which is crucial for resource exploration and development.
The model can be futher inferred for various applications, such as predicting lithology during drilling operations, optimizing well placement, and enhancing reservoir characterization.
